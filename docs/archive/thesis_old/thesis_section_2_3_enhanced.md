# 2.3 储能调度优化方法（增强版）

## 2.3.1 固定阈值策略

固定阈值策略是储能调度最简单的规则型方法。策略由两个参数定义：充电价格阈值θ_charge和放电价格阈值θ_discharge（要求θ_charge < θ_discharge）。在每个决策时刻t，其动作选择规则如式[eq:threshold-policy]所示：

```
a_t = charge,   if P_PV(t)>0, p(t)<θ_charge, SOC_t<SOC_max;
       discharge, if p(t)>θ_discharge, SOC_t>SOC_min;
       idle,      otherwise
```

其中，P_PV(t)表示t时刻预测光伏出力，p(t)表示t时刻电价，SOC_t表示电池荷电状态。

该策略的优势是实现简单、可解释性强，但策略质量高度依赖于阈值参数与电价分布的匹配程度。若放电阈值设定过高（高于样本电价上界），则系统在整个仿真周期内都不会放电。

## 2.3.2 24小时前瞻滚动优化

滚动优化（Rolling Horizon Optimization）在每一决策时刻t求解一个包含未来24小时预测信息的优化子问题，但只执行首小时的决策动作，然后在t+1时刻用最新的预测信息重新求解。该策略在计算复杂度和决策质量之间取得了折中。

在本系统中，滚动优化采用快速24小时前瞻价差策略：在每个决策时刻，考察未来24小时的价格序列，在有正价差的时段优先安排充放电动作，并计入循环成本、短缺惩罚和SOC终端稳定惩罚，通过滚动步进实现对全年小时级数据的完整调度。

## 2.3.3 神经网络策略蒸馏

策略蒸馏（Policy Distillation）[17]的核心思想是将复杂教师模型的策略知识迁移到轻量级学生神经网络中，使得学生网络能够以极低的推理成本模仿教师的决策行为。在本系统中，教师模型为2.3.2节所述的24小时前瞻滚动优化器，其每个决策时刻的求解本质上是求解一个带约束的线性规划子问题；学生模型为一个参数量仅约3万的前馈神经网络，推理仅需一次矩阵乘法前向传播，无需迭代求解。

### 2.3.3.1 问题建模与输入特征设计

策略蒸馏被建模为一个监督学习问题：以滚动优化器在每个决策时刻t的首步调度动作为监督标签，训练神经网络学习从系统状态到调度动作的映射f_θ: R^77 → A。

输入特征维度d=77，具体构成如下：

- **电池状态**（1维）：当前荷电状态SOC_t。
- **时间编码**（4维）：小时和月份的sin/cos周期编码，使网络感知日电价峰谷和季节性变化。
- **前瞻预报序列**（72维）：未来24小时逐时光伏出力预测{P_PV(t+k)} (k=0..23)、电价序列{p(t+k)} (k=0..23)、负荷序列{L(t+k)} (k=0..23)。这三组序列赋予网络与教师模型等价的24小时前瞻视野。

### 2.3.3.2 单阶段回归策略的局限性（Stage20）

最初的策略网络设计采用直接回归架构：一个两层MLP（77→128→128→2，ReLU激活，Dropout=0.1），同时输出充电功率和放电功率两个连续值。

训练采用70/15/15的时间顺序划分（训练/验证/测试），损失函数为Smooth L1 Loss（Huber损失），优化器为AdamW（学习率10⁻³，权重衰减10⁻⁴），批量大小为256，早停耐心值为10个epoch，梯度裁剪阈值为2.0。

然而，实验揭示了一个结构性问题：该模型的方向分类准确率仅为0.4017，低于多数类基线0.5017（即始终预测"空闲"的准确率）。其混淆矩阵为：

```
        pred_idle  pred_charge  pred_discharge
idle       975        256           480
charge      84        379          1418
discharge    4          1           152
```

其中行表示真实标签[空闲, 充电, 放电]，列表示预测标签。可见模型将大量真实放电样本误分类为充电（1418个），这在物理上是严重的错误——将本应收敛收益的放电动作逆转为消耗电量的充电动作。

**根本原因**：单阶段回归将"做什么动作"和"动作幅度多大"两个不同质的决策耦合在同一个输出空间中。回归损失对小幅功率误差的惩罚远小于方向性错误的实际代价，使得模型倾向于输出模糊的小幅值，而非做出明确的方向判断。

### 2.3.3.3 两阶段解耦策略（Stage20B）

针对单阶段回归的失败模式，提出了两阶段解耦架构，将调度决策分解为先后两个子任务：

**第一阶段：动作方向分类。** 将教师策略在每个时刻的决策映射为三种离散动作类型——充电（charge）、放电（discharge）和空闲（idle）。判定规则为：若教师计划的充电功率P̂_charge > ε，标记为充电；若P̂_discharge > ε，标记为放电；否则标记为空闲。其中ε=5.6×10⁻³ kW为动作判定阈值。

**第二阶段：条件功率回归。** 对第一阶段判定为充电或放电的时刻，由对应的子网络预测归一化功率值（充电功率/最大充电功率，或放电功率/最大放电功率），通过Sigmoid激活函数将输出限制在[0, 1]区间。

网络架构采用共享编码器+多任务头结构：

- **共享编码器**：77→128→64，ReLU激活，Dropout=0.1，提取系统状态的通用表征。
- **方向分类头**：Linear(64→3)，输出三类logits。
- **充电功率头**：Linear(64→1) + Sigmoid，输出归一化充电功率。
- **放电功率头**：Linear(64→1) + Sigmoid，输出归一化放电功率。

总参数量约3万，远小于典型深度强化学习网络的规模，推理延迟在毫秒级。

### 2.3.3.4 复合损失函数设计

两阶段架构的损失函数由三项组成，分别对应不同的优化目标：

```
L = L_dir + 0.5·(L_charge + L_discharge) + 1.0·L_opposite
```

**方向分类损失 L_dir**：采用加权交叉熵损失（Weighted Cross-Entropy）。由于三类动作样本严重不均衡（空闲样本占多数），使用逆频率类别权重w_c = N / (3·N_c)，其中N为总样本数，N_c为类别c的样本数。权重被截断至[0.25, 10.0]以防止极端值主导训练。

**条件功率回归损失 L_charge 和 L_discharge**：采用Smooth L1 Loss，但仅在真实标签为对应方向时才计入。即充电损失仅计算真实动作为充电的样本，放电损失同理。这使得功率头不会被无关样本干扰。

**反向惩罚项 L_opposite**：

```
L_opposite = (1/|S_c|)·Σ_{i∈S_c} p_i^(discharge) + (1/|S_d|)·Σ_{i∈S_d} p_i^(charge)
```

其中S_c和S_d分别为真实标签为充电和放电的样本集合，p_i^(a)为softmax后动作a的预测概率。该项直接惩罚充电时刻预测为放电（以及反向）的错误概率，因为充放逆转是代价最高的方向错误——它将系统收益方向完全翻转。

### 2.3.3.5 训练与评估

训练配置与单阶段模型保持一致：AdamW优化器（学习率10⁻³，权重衰减10⁻⁴），批量大小256，梯度裁剪阈值2.0，早停耐心值10个epoch。数据按时间顺序70/15/15划分，输入特征使用训练集统计量进行Z-score标准化。

模型评估包含两个维度：

1. **开环测试集**上的分类与回归指标；
2. **闭环物理回放**——将模型部署到与教师完全相同的仿真环境中，按模型输出动作更新SOC并计入物理约束（SOC边界、PV侧充电上限、并网容量），计算全周期收益并与教师对比。

### 2.3.3.6 实验结果

两阶段策略的关键指标如下表所示：

| 指标 | Stage20（回归MLP） | Stage20B（两阶段） |
|------|-------------------|-------------------|
| 方向准确率 | 0.4017 | **0.9908** |
| 多数类基线 | 0.5017 | 0.5073 |
| Macro-F1 | 0.3806 | **0.9749** |
| 放电召回率 | 0.9682 | **0.9804** |
| 放电→充电误分类 | 1 | **0** |
| 充电MAE (kW) | 0.0233 | 0.2197 |
| 放电MAE (kW) | 0.0204 | 0.0084 |
| 收益保留率 | 83.8% | **131.7%** |

两阶段架构的方向混淆矩阵接近对角矩阵：

```
        pred_idle  pred_charge  pred_discharge
idle       1164         2            11
charge        9      1308             0
discharge     2         0           100
```

**关键发现**：

1. **方向决策解耦是核心突破**。将方向分类与功率回归分离后，方向准确率从0.4017跃升至0.9908，显著超越多数类基线。充电↔放电的致命性误分类降至零。

2. **功率预测精度存在取舍**。充电MAE从0.0233上升至0.2197 kW，这是因为两阶段模型中功率头仅在其对应方向样本上训练，有效样本量减少。但方向正确前提下的功率误差对收益影响远小于方向错误。

3. **闭环收益保留率达131.7%**。两阶段策略在同窗口测试中的增量收益（0.0768 EUR）超过了教师策略（0.0583 EUR）。这并非"学生超越教师"，而是因为教师策略的滚动优化包含对未来24小时点预测的过度依赖，而学生网络通过学习大量样本中的统计规律，在某些场景下做出了更稳健的决策。

### 2.3.3.7 方法总结

本系统的策略蒸馏方法呈现一条清晰的技术演进路径：单阶段回归（失败）→ 两阶段解耦（成功）。核心经验是：对于包含离散决策边界（充/放/闲）和连续参数（功率大小）的混合动作空间，将分类和回归解耦为共享特征编码器上的独立任务头，并设计针对性损失函数（逆频率加权+条件回归+反向惩罚），显著优于单阶段直接回归。该方法不使用强化学习，通过监督学习即可实现对显式优化策略的有效蒸馏，避免了策略梯度方法在储能场景中的稀疏奖励和样本效率问题。
