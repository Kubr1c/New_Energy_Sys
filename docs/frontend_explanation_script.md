# 前端系统功能讲解稿（论文答辩检查稿）

## 1. 前端系统定位

新能源储能侧优化调度系统的前端部分面向光伏功率预测、储能调度仿真和多情景收益分析过程，承担实验结果展示、交互参数设置和指标解释功能。系统前端并不直接给出工程建设结论，而是将数据质量、预测模型、预测验收、储能调度和配置敏感性分析集中到统一界面中，为论文中的方法验证和系统实现说明提供支撑。

系统前端的主要功能关系如下：

```mermaid
flowchart LR
  A["数据质量与特征检查"] --> B["模型误差比较"]
  B --> C["预测结果验收"]
  C --> D["天气与电价驱动的调度仿真"]
  D --> E["储能动作与并网影响展示"]
  D --> F["多收益情景对比"]
  F --> G["配置敏感性分析"]
```

## 2. 登录与公共布局

系统入口为登录页，页面显示系统名称“新能源储能调度系统”，并说明系统围绕光伏预测、储能调度与策略评估展开。用户完成身份验证后进入主界面。登录功能主要用于保证内部数据页面在受控状态下访问。

主界面采用左侧导航和顶部状态栏布局。左侧导航包含系统总览、储能调度、预测分析、配置敏感性、数据管理和预测验收六个页面。顶部区域显示当前页面标题、时间和用户状态。前端页面同时设置加载、错误和空数据三类状态，用于处理接口读取中、接口异常和数据未生成等情况，避免页面在数据缺失时出现无解释的空白结果。

## 3. 系统总览页面

系统总览页面用于概括系统的数据、模型和调度结果。该页面不是单独的实验页面，而是对各模块核心信息的集中呈现。

页面顶部说明系统面向光伏配储调度场景，集成天气驱动仿真、退化成本评估和多收益情景量化展示。该部分的重点是说明系统研究对象和展示边界，即系统关注的是“有储能方案相对于无储能基准的增量效果”。

总览页的指标卡片包括最优情景净增量、正净增量情景数和预测输入模型。最优情景净增量用于显示当前多收益情景中表现最好的结果；正净增量情景数用于说明在全部展示情景中有多少情景取得正净增量；预测输入模型用于说明当前预测曲线所采用的模型来源。

“预测效果概览”图展示实际光伏功率与预测光伏功率之间的关系。该图支持日期切换、预测时长选择和天气场景筛选。预测时长包括 t+1h、t+6h 和 t+24h，分别表示提前 1 小时、6 小时和 24 小时预测。该图用于说明预测模型是否能够较好跟随光伏出力变化。由于预测结果是储能调度的输入，预测偏差会影响后续调度策略。

右侧“预测输入品质”区域展示当前展示模型、测试集最优模型、测试集 nRMSE、数据记录数、工程特征数和地理位置。其中，当前展示模型用于本页曲线展示，测试集最优模型来自模型排行榜，两者属于不同展示口径，不应直接混同。

“调度数据口径”区域说明系统使用 PVDAQ、NSRDB、OPSD、Open-Meteo 等公开数据或代理场景。该区域用于明确收益结果为参考仿真结果，Rawhide 相关内容为公开容量参数参照场景，不构成实测电站运行数据或真实市场结算结果。

## 4. 储能调度页面

储能调度页面是系统前端的核心展示页面。该页面包括天气驱动实验台、调度结论、收益与退化、配置与策略四个页签。系统默认进入天气驱动实验台，以便首先展示参数可调的储能调度仿真过程。

### 4.1 天气驱动实验台

天气驱动实验台用于展示在给定天气预测、电价场景和储能参数条件下，系统如何生成储能充放电方案，并同步计算收益、并网功率、SOC 和退化指标。该页面体现了光伏预测结果进入储能调度的过程。

参数设置区包含调度日期、调度时段、天气场景、电价场景、优化目标、优化算法、储能容量、储能功率、充电效率、放电效率、初始 SOC、SOC 下限和 SOC 上限。

调度日期用于确定仿真起始时间。调度时段可选 24h、48h 和 72h，表示系统向后考虑的调度窗口长度。天气场景包括实时天气预报、晴天、多云、阴天和自定义场景，用于改变辐照度、云量、温度等输入条件。电价场景包括峰谷分时电价、固定电价、高波动电价、合成电价和光伏鸭形曲线电价，用于比较不同价格结构下的储能收益。

优化目标包括经济性优先、平滑优先和综合优化。经济性优先侧重收益结果，平滑优先侧重降低并网功率波动，综合优化在收益和平滑之间进行折中。优化算法包括规则策略、滚动优化和多目标优化。规则策略便于解释调度逻辑，滚动优化体现逐时更新思想，多目标优化用于同时考虑多个评价目标。

储能容量以 kWh 为单位，表示电池可存储的电量；储能功率以 kW 为单位，表示单位时刻最大充放电能力。充放电效率表示能量转换损失。初始 SOC 表示仿真开始时电池电量比例，SOC 上下限用于约束电池运行范围，避免过充或过放。当前默认参数采用 24h 调度窗口、高波动电价、4000 kWh 容量、2000 kW 功率、0.95 充放电效率和 0.10-0.90 的 SOC 运行区间，用于提供较稳定的演示基准。

“光伏出力与并网影响”图是天气驱动实验台的主结果图。上图展示光伏预测功率和调度后并网功率，下图展示储能净功率。储能净功率为放电功率减去充电功率，正值表示储能放电支撑并网，负值表示储能充电吸收光伏出力。该图的核心关系为：

并网功率 = 光伏出力 - 储能充电功率 + 储能放电功率。

该展示方式避免将电站级光伏出力与设备级充放电功率直接并列比较。光伏出力表示电站侧可产生的功率，储能净功率表示储能设备的调节动作，二者对象不同、量级不同。通过上下分图展示，可以说明储能虽然不一定与光伏出力同量级，但仍会改变最终并网功率曲线。

SOC 图用于展示电池荷电状态在调度窗口内的变化。若 SOC 接近上限，说明电池继续吸收光伏出力的空间不足；若 SOC 接近下限，说明继续放电的空间不足。该图用于解释储能动作受到电池状态约束的原因。

电价收益图用于展示电价变化与储能收益之间的关系。储能经济性并不只取决于充放电次数，而取决于放电收益能否覆盖充电成本、效率损失和电池退化成本。因此，高波动电价通常更有利于形成正净增量收益。

方案对比图和表格用于比较无储能基准、规则策略、滚动优化和多目标优化等方案。无储能基准作为对照组，其他方案的增量收益均相对于该基准计算。若增量收益为正，表示当前策略在该情景下优于无储能基准；若为负，则表示未超过无储能基准。

天气信息区域展示选中调度时刻的天气预报，包括温度、云量、辐照度、风速和湿度。该区域与图表选中时间联动，时间按照电站当地时间显示。天气趋势图展示调度窗口内天气变量变化，用于解释光伏出力预测曲线的变化原因。光伏出力预测曲线则展示天气输入转换为光伏功率输入的结果。

右侧 KPI 摘要分为经济性、运行效果和电池健康三类。经济性指标展示总收益和增量收益；运行效果指标展示削峰效果和平滑效果；电池健康指标展示等效循环次数和 SOH 变化。等效循环次数用于衡量电池使用强度，SOH 变化用于反映电池健康衰减情况。

参数灵敏度分析用于展示容量或功率按比例变化时，增量收益的变化趋势。该图用于说明当前情景下收益对容量或功率变化的敏感程度。运行历史表保存最近调度记录，可用于复盘不同参数组合下的结果。

### 4.2 调度结论

调度结论页签展示多收益情景的结果，用于说明不同经济假设下储能调度收益的变化。顶部指标包括最优情景净增量、正净增量情景数和情景类型数。其中，正净增量情景数和情景类型数可进一步查看对应明细。

“净增量收益（按情景）”图以柱状图形式展示各情景的净增量收益。柱状图为正表示该情景相对于无储能基准取得正增量，为负表示未形成正增量。该图用于直观比较不同经济条件对储能收益的影响。

情景展示表列出情景名称、情景类型、毛增量收益、退化成本、额外收益、净增量收益、SOH 终点、等效满循环和边界说明。展开行显示策略口径、情景解释、更换成本、循环寿命倍数、年历衰减、容量价值和约束状态。界面不再直接展示内部配置 ID，以避免将实现层编号暴露给展示对象。

情景说明区域对每个情景进行解释。基准纯套利情景用于提供无额外收益和默认退化参数下的参照结果；价格波动增强情景用于观察价差扩大后套利收益是否能够覆盖退化成本；容量价值低档和高档情景用于分析容量价值叠加对净增量收益的影响；低退化成本情景用于观察电池成本改善后的收益边界；最优纯套利情景用于展示更有利成本条件下纯套利组合的表现；退化约束主动循环情景用于说明退化惩罚对策略行为的影响；激进策略对照情景用于说明忽略退化成本可能导致净收益恶化。

### 4.3 收益与退化

收益与退化页签展示参考电站仿真和电池退化成本评估结果。页面顶部列出参考电站的光伏容量、储能配置、投运年份和仿真周期。该部分仅作为公开容量参数参照场景。

收益卡片和增量收益对比图展示不同策略相对于无储能基准的收益变化。图表金额单位为元，口径为基于 OPSD 映射或项目代理电价并按固定汇率换算后的仿真增量收益。部分结果区分是否扣除退化成本，用于说明电池老化成本对经济性评价的影响。

电池退化成本评估展示退化成本、SOH 起止值和等效满循环次数。该区域说明储能调度不能只考虑短期充放电收益，还需要将电池寿命损耗纳入评价。

### 4.4 配置与策略

配置与策略页签展示储能配置敏感性和策略评价结果。储能配置敏感性表列出不同容量、功率配置下的增量收益和候选边界。候选边界表示当前仿真条件下收益和运行强度相对均衡的配置，不表示实际工程配置结论。

策略评价区域展示各策略的评分、策略类型、总调度收入、增量收益、循环次数和平均 SOC。策略评分对比图展示经济性、约束满足和风险控制等评分项。三维评分雷达图展示策略评分结构，分值越高表示评价越好。需要说明的是，策略评分雷达图与模型误差雷达图方向不同，前者越外表示评分越高，后者越靠近中心表示误差越低。

## 5. 预测分析页面

预测分析页面用于比较不同光伏功率预测模型在测试集上的误差表现。该页面的作用是说明调度输入模型的选择依据。

测试集模型排行榜默认展示同一预测时长内的 Top10 记录。页面提供预测目标/时长筛选和特征集筛选。预测时长包括 t+1h、t+6h 和 t+24h。不同预测时长对应不同预测难度，因此排行榜只在同一预测目标内比较，不将不同预测时长直接混排。

表格指标包括 nRMSE、MAE、RMSE 和日间 nRMSE。nRMSE 为归一化均方根误差，MAE 为平均绝对误差，RMSE 为均方根误差，日间 nRMSE 只关注太阳高度满足条件的日间样本。上述指标均为误差指标，数值越低表示模型表现越好。

“测试集 nRMSE 对比”柱状图使用与排行榜相同的 Top10 数据，横轴采用局部范围以放大模型误差差异。因此，图形差异用于辅助观察，具体数值仍应以表格为准。

“多指标误差雷达图”展示当前筛选结果中前 5 个模型的误差结构。各轴均为误差指标，越靠近中心表示误差越低。该图用于观察某个模型是否只在单一指标上较好，或在多项误差指标上较均衡。

## 6. 配置敏感性页面

配置敏感性页面用于分析储能容量和功率变化对仿真收益的影响。该页面展示的是敏感性仿真结果，而不是实际工程配置结论。

“储能配置候选边界分析”散点图以增量收益和循环次数为主要维度。横轴越靠右表示增量收益越高，纵轴越高表示电池循环强度越大。高亮点表示在当前仿真中收益和循环强度相对均衡的候选配置。

“容量 x 功率增量收益热力图”展示不同容量倍率和功率倍率组合下的收益水平。颜色仅表示当前仿真结果中的相对收益高低，不代表跨项目或跨数据集的绝对配置结论。

明细表列出仿真配置、容量倍率、功率倍率、仿真增量收益、循环次数和候选边界。表格将原始配置编号转化为容量倍率、功率倍率和目标组合等可解释字段，便于将图中结果对应到具体参数。

## 7. 数据管理页面

数据管理页面用于说明系统输入数据的质量状态、特征重要性和后端任务运行情况。该页面包括预测特征重要性和调度收益构成两个展示模式。

顶部数据质量指标包括样本数量、缺失字段数和数据覆盖率。样本数量表示清洗后的数据记录数；缺失字段数表示质量报告中存在缺失的字段数量；数据覆盖率表示目标时间范围内样本的覆盖程度。

“清洗字段完整性检查”来自清洗质量报告，主要检查字段缺失、时间字段状态、样本时间覆盖率、目标字段完整性和特征字段完整性。该区域反映清洗数据的完整性，不等同于对所有工程特征逐项重新计算。

“预测特征重要性 Top20”展示当前模型内部的特征重要性分数。该分数表示各输入变量在当前模型中的相对贡献，不用于跨模型直接比较。图中使用中文字段名，原始字段名保留在提示信息中，用于排查数据来源。

“模型训练 / 调度仿真”区域列出可触发的后端任务，包括模型训练、模型比较、预测刷新、调度仿真、策略比较、滚动优化、策略评分和敏感性分析等。任务提交后，前端显示任务状态、开始时间、耗时、最后更新时间和错误详情。最近任务表用于复盘任务运行记录。

“调度收益构成对比”展示多收益情景下毛增量收益、额外收益和退化成本的构成。退化成本作为负向贡献展示，用于解释净增量收益为正或为负的来源。对应表格列出收益情景、情景类型、净增量收益、毛增量收益、退化成本、额外收益、SOH、约束状态和边界说明。

## 8. 预测验收页面

预测验收页面用于对指定日期或时间窗口内的预测结果进行细粒度检查。与系统总览中的预测图不同，该页面提供更多筛选条件和指标汇总，用于检验预测模型在不同日期、预测时长和天气场景下的表现。

左侧控制面板包括日期选择、预测时长、预测方案、展示粒度和天气场景。展示粒度包括小时、日、3 日和 7 日。小时粒度适合观察逐时功率变化，日粒度和多日窗口适合观察累计发电量误差。

主图在小时粒度下展示实际功率与预测功率对比，并使用柱状图表示预测误差。红色表示预测偏高，蓝色表示预测偏低。切换为日、3 日或 7 日粒度时，图表展示实际发电量与预测发电量的对比。

底部指标汇总包括预测误差类、发电量类和基准对比类。预测误差类包括 MAE、RMSE、nRMSE 和 Bias；发电量类包括当前范围实际发电量、预测发电量和误差率；基准对比类包括持续性基准 RMSE 和相对基准提升率。当缺少有效样本时，界面显示“-”或“数据不足”，避免将无数据状态误解释为零误差。

## 9. 建议讲解顺序

答辩演示可按以下顺序展开。首先进入系统总览，说明系统对象、数据来源和结果边界。其次进入数据管理页，说明数据质量、字段完整性和特征重要性。然后进入预测分析页和预测验收页，说明模型误差比较与预测曲线验收。之后进入储能调度页，在天气驱动实验台中调整参数并运行调度，重点解释光伏出力、储能净动作和并网功率之间的关系。最后进入调度结论和配置敏感性页面，说明不同经济情景和容量功率组合对净增量收益的影响。

## 10. 常见问题说明

增量收益为负的原因在于储能调度存在效率损失和电池退化成本。当电价价差不足、额外收益缺失或循环退化成本较高时，充放电带来的毛收益可能无法覆盖成本，因此净增量收益为负。

光伏预测功率和储能充放电功率量级不同，是因为二者对应的对象不同。光伏预测功率表示电站侧出力，储能充放电功率表示储能设备的调节动作。储能不需要与光伏电站同量级，也可以通过局部充放电改变并网功率曲线。

并网功率是指最终送入电网侧的功率。储能充电会吸收一部分光伏出力，使并网功率低于光伏出力；储能放电会向电网侧补充功率，使并网功率高于单独光伏出力。

SOH 表示电池健康状态。储能调度若频繁循环，会带来电池寿命损耗。因此系统同时展示收益和 SOH，用于说明经济性评价需要考虑退化成本。

t+1h、t+6h 和 t+24h 表示不同提前量的预测任务。提前时间越长，预测难度通常越高，因此不同预测时长应分别比较。

配置敏感性结果不能直接表述为工程建设建议。该结果来自当前数据和参数假设下的仿真扫描，只能说明相对收益变化和候选边界，真实工程配置还需要结合投资成本、安全规范、接入条件和长期运行约束。
